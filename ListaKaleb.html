Configuración de Custom Widget

CardView para Sketchware Pro
androidx.cardview:cardview:1.0.0
CardViewWidget
androidx.cardview.widget.CardView
Selecciona Layout (ya que funciona como un contenedor para meter otros elementos dentro).
Selecciona AndroidX (o Containers según las categorías que tengas creadas).
Copia y pega la siguiente configuración base para las sombras y esquinas:
app:cardCornerRadius="8dp"
app:cardElevation="4dp"
app:cardUseCompatPadding="true"
